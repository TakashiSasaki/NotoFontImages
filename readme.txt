Google Noto (no more tofu) Fonts covers all scripts encoded by Unicode. This repository provides PNG images of glyphs over almost all codepoints. Currently all PNG files are generated in 40pt.

Usage:
https://takashisasaki.github.io/NotoFontImages/40pt/8_digit_lower_case_hex_of_codepoint.png

Example:
https://takashisasaki.github.io/NotoFontImages/40pt/0001f63c.png

Noto is a trademark of Google Inc. Noto fonts are open source. 
All Noto fonts are published under the SIL Open Font License, Version 1.1. 
Language data and some sample texts are from the Unicode CLDR project.

Note:
Since this repository is not official one, please let me ( https://twitter.com/TakashiSasaki ) know i I need to delete it. I don't want to make wrong use of these fonts and also want to prevent abuses of intellectual property rights.
